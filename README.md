# 📺 YouTube UI Clone (HTML & CSS Only)

A high-fidelity visual clone of the YouTube homepage, built to demonstrate core front-end layout skills without the use of JavaScript. 



## 🚀 Overview
The goal of this project was to replicate the complex UI of YouTube, focusing on **responsive design** and **component-based CSS**. This project showcases the ability to build modern, professional interfaces using only semantic HTML and modern CSS techniques.

## 🛠️ Tech Stack
* **HTML5:** Structured with semantic tags (`<nav>`, `<main>`, `<aside>`) for better SEO and accessibility.
* **CSS3:** Leveraged advanced positioning and styling.
* **Flexbox/Grid:** Used for the responsive video thumbnail grid and the navigation header.

## ✨ Key Features
* **Top Navigation Bar:** Fixed-position header with search bar, icons, and user profile placeholder.
* **Responsive Sidebar:** A sidebar containing navigation links that remains consistent with the YouTube branding.
* **Video Grid:** A fluid grid system that adjusts the number of video thumbnails based on the screen width.
* **Hover Effects:** Interactive CSS states on video thumbnails and sidebar links to mimic the real YouTube user experience.
